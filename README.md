# AirB

- 프로젝트 사이트: **Airbnb**
- 소개: 세계 최대의 숙박 공유 서비스이며, 자신의 방이나 집, 별장 등 사람이 지낼 수 있는 모든 공간을 공유할 수 있는 플랫폼입니다.

<br/>

## 프로젝트 선정 이유
- 다양한 숙박 공유 서비스 중, 가장 직관적인 UI와 User Friendly 한 환경을 제공합니다.
- 폭넓은 숙박 필터링 검색을 제공하며, 사용자가 원하는 위치, 날짜, 테마, 가격, 시설 등을 선택해서 최적화된 결과를 보여줍니다.

<br/>
